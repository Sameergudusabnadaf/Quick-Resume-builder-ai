# Local AI Resume Generator

A powerful, entirely local toolkit designed to help you quickly generate clean, professional FAANG-style resumes without relying on external AI APIs.

This project provides two distinct ways to build your resume:

1. **Command Line Interface (CLI):** An interactive Python script that asks you for details step-by-step and generates a text-based resume.
2. **Web UI:** A sleek HTML/CSS/JavaScript web application where you can fill out a form (or use "Magic Prompt" text extraction) and instantly download a `.doc` version of your resume.

---

## ✨ Features

- **100% Local Processing:** No external APIs required. Your data never leaves your computer.
- **FAANG-Style Formatting:** Generates clean, ATS-friendly resume layouts optimized for top tech companies.
- **Interactive CLI Tool:** Provides a step-by-step terminal wizard to collect your information quickly.
- **Modern Web Interface:** A beautifully designed frontend with interactive forms and instant resume previewing.
- **Word Document Export:** Click one button in the web interface to instantly download a `.doc` file that you can further edit in Microsoft Word.
- **Smart Parsing:** Basic rule-based extraction for converting your inputs into a polished layout.

---

## 🛠️ Requirements

### For the Python CLI (`resume_generator.py`)

- **Python 3.x:** Ensure Python is installed on your system.
- **No external packages required!** It uses pure Python built-in libraries (`os`, `sys`).

### For the Web Application

- **A Modern Web Browser:** Chrome, Firefox, Edge, Safari, etc.
- **(Optional) Local Web Server:** Python's built-in `http.server` is recommended for running the HTML files locally.

## 🚀 How to Install and Run (Step-by-Step Guide)

### Prerequisites (Before you begin)

1. **Download the Project:** Download or clone this project folder to your local computer.
2. **Install Python:** If you don't have Python installed, download it from [python.org](https://www.python.org/downloads/) and install it. **Important:** Make sure to check the box that says "Add Python to PATH" during the installation process!

---

### Method 1: Running the Command Line (CLI) Resume Builder

**Step 1:** Open your computer's Terminal (Mac/Linux) or Command Prompt / PowerShell (Windows).
**Step 2:** Navigate into the project folder using the `cd` (change directory) command. For example:

```bash
cd c:\Users\samee\OneDrive\Desktop\sir
```

_(Replace the path above with the actual path where you saved the folder)._

**Step 3:** Start the Python generator script by typing the following command and pressing Enter:

```bash
python resume_generator.py
```

**Step 4:** The script will welcome you and ask for your details step-by-step (Full Name, Email, Skills, Experience, etc.). Type your answers and press Enter for each one. (If you want to skip an optional field, just press Enter without typing anything).

**Step 5:** Once finished, your resume will be generated instantly! You can find the finished file saved in the same folder as a text document (e.g., `yourname_resume.txt`).

---

### Method 2: Running the Interactive Web Application

**Step 1:** Open your computer's Terminal or Command Prompt.
**Step 2:** Navigate into the project folder:

```bash
cd c:\Users\samee\OneDrive\Desktop\sir
```

**Step 3:** Start a local web server by running this exact command:

```bash
python -m http.server 8000
```

_(You should see a message in your terminal saying it's serving HTTP on port 8000)._

**Step 4:** Open your favorite web browser (Chrome, Edge, Firefox, Safari, etc.).
**Step 5:** In the address bar at the top, type the following URL and press Enter:

```text
http://localhost:8000
```

**Step 6:** You will now see the sleek Web UI! Fill in the form fields with your personal and professional details.
**Step 7:** Click **"Generate Resume"** at the bottom to see an instant preview of your resume.
**Step 8:** Click **"Download .doc (Word)"** to save your beautifully formatted resume directly to your computer as a Microsoft Word document!
